Add it to Gmail / Google Workspace

  1. Click Copy signature above.
  2. In Gmail, open Settings (gear icon) → See all settings → General tab.
  3. Scroll to Signature → Create new, name it, then click in the box and paste (Ctrl/Cmd + V).
  4. Set it as the default for New emails and Replies, then Save Changes at the bottom.

Why this one won't hurt your deliverability

  • Table layout, inline styles — the only formatting email clients render reliably.
  • Mostly text, one small logo — image-heavy signatures are a spam signal; this stays text-first and reads fine even if the image is blocked.
  • Honest links — the visible text matches where it goes (no hidden or shortened URLs, no tracking pixel).
  • No spam-trigger words, no ALL-CAPS, no exclamation-heavy copy.
